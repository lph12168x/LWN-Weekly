## ✨ 特性

- 🌙 **亮色/暗色主题** - 支持一键切换，自动记忆偏好
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 📋 **目录导航** - 每期文章自带目录，快速跳转
- 🎨 **美观排版** - 精心设计的排版和色彩方案
- 🔍 **信息框** - 支持笔记、警告、成功等多种信息框样式
- 📊 **数据表格** - 内核开发数据清晰呈现

## 📋 内容范围

每期翻译内容包括：

- **内核开发** - Linux 内核最新特性和开发统计
- **文件系统** - 文件系统相关的深度文章
- **发行版** - 各大 Linux 发行版的最新动态
- **安全** - 开源软件安全相关资讯
- **开发工具** - 编程语言、工具链的更新
- **简讯** - 一周社区简讯汇总

## 🤖 自动化更新

本项目通过 GitHub Actions 实现自动化更新：

- ⏰ **每周一 10:15** 自动抓取 LWN 最新免费文章
- 📝 自动翻译并生成中文 HTML 页面
- 🔄 自动更新索引页面
- 📤 自动推送到本仓库
